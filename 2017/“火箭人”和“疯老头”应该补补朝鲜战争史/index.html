“火箭人”和“疯老头”应该补补朝鲜战争史

发表于 2017-09-26 | 分类于 时事

一个差不多已被击败的敌人，正像僵尸一般，从美国模糊的战争记忆坟墓中爬出来。

朝鲜欢欣鼓舞地发射导弹，令其飞越日本，在太平洋上溅起一朵朵水花。凭借惊人的技术手段，它正在建造一个或许很快就能将氢弹投到洛杉矶、芝加哥甚至是曼哈顿的武器系统。

与此同时，两位发型怪、脸皮薄的新手领袖开始用离奇的方式互相侮辱。特朗普总统说金正恩是“火箭人”，并威胁要“彻底摧毁朝鲜”。金正恩说特朗普是“精神错乱的美国疯老头”，威胁要以“火”来“驯服”他。

他们当中至少有一个应该更明智一点。

有了装载核弹头的洲际导弹所带来的威慑力，朝鲜领导人似乎很有机会延续父亲和祖父的做法——度过大权独揽的一生，直至晚年因自然原因而死亡。然而，独裁者的DNA告诉金正恩，同美国发生全面战争是有风险的。上一次发生这种事的时候，局面对他的家人来说很不利。朝鲜战争(1950-53)期间，他的祖父——伟大领袖金日成——蜷缩在掩体里，与此同时美军炸平了他的城市，他的人民成批地死去。

这应该让美国的决策者们明白——特别是我们这位历史知识很成问题的总统，以及他那些高呼“鲜血与土地”的支持者们——朝鲜不太可能主动发起进攻，除非金氏政权被激怒（也许是由于清晨发出的一条极力鼓吹战争的推文），相信自己的生存真的处于危险。特朗普政权应该考虑金氏家族的历史。它是一个专注长期生存的犯罪团伙，在奴役人民方面的水平，远高于发起妄自尊大的战争。

可悲的是，美国在很大程度上忘记了朝鲜战争的教训，尽管那场冲突导致了逾3.3万名美国参战人员丧生。这种集体健忘有着多种原因：朝鲜战争最终以不体面的平局告终，以至于根本无法庆祝。它没有制造关于“最伟大的一代”的神话，也没有被拍成什么令人难忘的电影。然后，越南战争爆发了，那是第一场真正通过电视播报的战争，至今公共电视台仍在对之进行分析。越南战争融入了我们的文学与电影文化，令朝鲜战争相形失色，成了“被遗忘的战争”。

随着全球焦虑日益增加，是时候补上这段历史了。

1950年夏天，当朝鲜在苏联支持下发起朝鲜战争时，金日成只有38岁，是一个任性、好斗、缺乏经验的独裁者，和他如今掌权的孙子有点像。入侵之前与斯大林举行秘密会晤时，他就战争进程做了大量荒唐可笑的狂热分析，声称自己的军队将会踏平韩国。

他预言，一股强大的亲共游击力量将在韩国自发地涌现出来，与朝鲜军队并肩作战。后来这种事并没有发生。他还许诺说，韩国人民会团结在他的领导之下。后来他们也没有这样做。他得出一大堆不可靠的结论，最后向斯大林保证，朝鲜的胜利将在三天内到来，美国人也不会介入。然而战争至今尚未结束；美国仍在朝韩边境的非军事区巡逻。

在莫斯科郊外的别墅里，斯大林并没有完全相信金正日的这一套。他警告自己这位迫不及待的朝鲜追随者：“如果你被迎面痛击，我可不会伸一个手指头帮忙。”但是这位苏联老大哥想让美国难受，于是批准并支持金正日入侵韩国，并且要金日成做得看上去好像是韩国先动手挑起战争。

当然，美国发起了反击。哈里·S·杜鲁门总统(Harry S. Truman)、国会和公众对这场入侵感到愤怒，视其为对美国国格的挑战。不到一周，杜鲁门就批准动用地面部队。

战争一开始令人沮丧，美军迟疑不决，数千士兵丧生，之后美国凶残的战争机器开始以不可阻挡之势发动起来。美国空军使用炸弹和凝固汽油弹炸毁了朝鲜几乎所有人口中心。朝鲜战争时期的空中行动战略指挥官柯蒂斯·李梅(Curtis LeMay)将军估计，“在三年左右的时间里，我们杀死了——大致有——20％的人口。”也就是190万人。

美国部队——以及韩国和联合国军——击溃了朝鲜的入侵部队，占领了平壤，向北进军，抵达中国边界，实际上相当于消灭了朝鲜。毛泽东不愿忍受美军盘踞在自己家门口，于是加入战局。毛泽东手下的高级将领彭德怀一眼看出金日成是个战场上的白痴，说他的领导“极其幼稚”，于是将金日成挤出战争的领导链条，让他在自己发起的战争里只能无助地作壁上观。为了避免朝鲜因金正日铸成的大错而灭亡，中国军队死伤惨重，他们让金氏政权不至于沦为亚洲历史上无足轻重的脚注。

关于这段记载翔实、有据可查的历史，平壤的政治宣传者一直都在向朝鲜人民说谎，他们声称韩国和美国偷偷开启了这场战争，最后伟大领袖取得了辉煌胜利。但是伟大领袖的后人和他们的军事策划者心里清楚真相。

金氏家族的独裁政权尽管说了不少奥威尔式的胡言乱语，但它长久以来一直是靠着冷酷的理性才得以生存下来，即使是在它表现得狂怒好斗之时也是如此。

如果再来一场战争，金氏政权就必须认识到，如今它已不太可能从20世纪中叶那些共产主义旗舰国家那里得到巨大支持。弗拉基米尔·普京(Vladimir Putin)治下的俄罗斯只是斯大林苏联的一个匪徒版影子。中国的政治稳定取决于同西方开展活跃的贸易。此外，金正恩进行核试验与发射导弹所选择的时机非常过火，已经引起了中国国家主席习近平的反感。其结果就是朝鲜比以往任何时候都更加孤立——尽管它已成为全球的核威胁。

美国必须接受这个明显的事实：金正恩永远不会放弃他的导弹。但他知道，一旦使用它们，他就会灭亡，或是像他爷爷那样住在掩体里。他的核武器只有放在货架上时才能发挥最大效力。

特朗普应该裹住他的“烈火与愤怒”，不要再说那些被金正恩准确地描述为“史无前例的野蛮狂言”的言语。相反，华盛顿需要和金氏进行一场大范围的冷战：要有强大的军事准备，有力的间谍行动，灵活的制裁，与中国和俄罗斯的秘密谈判，并且乐于尽可能地同朝鲜人展开任何层次的对话。如果华盛顿做出单方面的姿态，将会有所帮助。可以接受一位朝鲜驻华盛顿大使。如有可能，还可以派出驻平壤大使。承认美军在朝鲜战争中对朝鲜城市的罪恶轰炸。尝试帮助朝鲜人民，让他们感觉似乎并不是整个世界都在和他们作对。

战争还是有可能到来，不做好准备将是美国的疏忽。金正恩究竟有多成熟，这一点值得怀疑。那些漂亮的武器会令他眼花缭乱，他可能会试图利用它们来控制整个朝鲜半岛。但是，鉴于他的家族在战争时期那段不自量力的惨痛历史，他还是明智一点为好。
